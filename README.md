# Get started

## Prepare

Create `data` folder (for sqlite):
```sh
mkdir data
```

Create `config` file:
```sh
cp config.example.yml config.yml
```

Install dependencies:
```sh
npm install
```

## Development

Run dev server:
```sh
npm run dev
```

## Production

Build the app:
```sh
npm run build
```

Run the app:
```sh
npm start
```

## Docker

Build the Docker container:
```sh
npm run docker
```
